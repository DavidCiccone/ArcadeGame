frontend-nanodegree-arcade-game
===============================

Clone of a classic frogger style game.

Navigate the player across the road to the water.  When the player hits the water it will reset and add on to the score counter.

Game will run in most modern browsers.

Technologies used: HTML5, CSS3, JavaScript, Canvas.

Game app is in file js/app.js

Art assets and engine provided by Udacity.
